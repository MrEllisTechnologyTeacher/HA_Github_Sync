# HA GitHub Sync Add-on

[![Home Assistant Add-on](https://img.shields.io/badge/Home%20Assistant-Add--on-blue)](https://www.home-assistant.io/addons/)

Sync and backup your Home Assistant configuration to a GitHub repository,
enabling agentic editing with GitHub Copilot.

See [DOCS.md](DOCS.md) for full setup and configuration documentation.
